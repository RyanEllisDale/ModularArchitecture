\chapter{Installation }
\hypertarget{_installation}{}\label{_installation}\index{Installation@{Installation}}
The modular architecture package can be added to your Unity project in three ways. The first two use the Package Manager, while the third is a manual fallback if the Package Manager is unavailable.\hypertarget{_installation_autotoc_md16}{}\doxysubsection{\texorpdfstring{Package Manager}{Package Manager}}\label{_installation_autotoc_md16}
You can access the Package Manager by opening your Unity project, selecting {\bfseries{Window}} from the top menu bar, and choosing {\bfseries{Package Manager}}. ~\newline
 In the top‑left corner of the Package Manager window, the {\bfseries{➕ (Add)}} button allows you to install external packages.

This package supports two Package Manager installation methods:\+


\begin{DoxyItemize}
\item {\bfseries{Add package from disk}} ~\newline

\item {\bfseries{Add package from Git URL}}
\end{DoxyItemize}

\hypertarget{_installation_autotoc_md17}{}\doxysubsubsection{\texorpdfstring{Package Manager Disk}{Package Manager Disk}}\label{_installation_autotoc_md17}
To install from disk, download and extract the package, place it anywhere on your machine, then select {\bfseries{Add package from disk}} and point Unity to the folder. Packages installed this way are {\itshape local} and not stored inside the project. ~\newline
 If the folder is moved or deleted, Unity will lose the reference and you’ll need to re‑add it.

 Open the Package Manager, click the {\bfseries{Add}} button, and select {\bfseries{Add package from Git URL}}. Unity will prompt you for a valid Git URL. ~\newline
 Use :\+ \href{https://github.com/RyanEllisDale/ModularArchitecture.git}{\texttt{https:\+/\+/\+github.\+com/\+\+Ryan\+Ellis\+Dale/\+\+Modular\+Architecture.\+git}}

\hypertarget{_installation_autotoc_md19}{}\doxysubsection{\texorpdfstring{Manual Install}{Manual Install}}\label{_installation_autotoc_md19}
Manual installation is also possible if the Package Manager is unavailable or not preferred. ~\newline
 To install the package manually, download and extract it, then place the extracted folder inside your project\textquotesingle{}s {\ttfamily Packages/\+} directory (located in the project root). If the {\ttfamily Packages/\+} folder does not exist, create it manually.

Ensure that you copy only the package’s root folder — the one that directly contains the {\ttfamily package.\+json} file.\hypertarget{_installation_autotoc_md20}{}\doxysubsection{\texorpdfstring{Thank You For Reading}{Thank You For Reading}}\label{_installation_autotoc_md20}
