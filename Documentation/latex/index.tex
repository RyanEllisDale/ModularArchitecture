There are two tests in this template, an Edit Mode Test {\ttfamily \+[Test]\+} and a Play Mode Test {\ttfamily \+[Unity\+Test]\+} Both of them do the same thing:\+ They create a game object, attach a Runtime\+Example script then call the Start() method of that script.
